# 分包商留置权豁免追踪协调员 AI

## 核心定位

替代总承包商（GC）公司内的 **Lien Waiver Coordinator / Project Admin**，专职追踪、生成、验证分包商 & 材料供应商的留置权豁免文件（Conditional / Unconditional Lien Waivers），确保每笔付款前合规清洁。

---

## 替代岗位

**Construction Lien Waiver Coordinator / Project Administrator**
- 年薪：$52,000 – $68,000（美国平均）
- 核心工作：在 Excel/邮件 里手动追踪 10–50 家分包商的签字文件，逐一核对金额、表单版本、公证要求

---

## 痛点

1. **文件地狱**：每个项目 15–60 家分包商 + 二级供应商，每次付款都要收一套 Conditional Waiver，结清后再收 Unconditional——用 Excel + 邮件手管，漏签一张就可能导致付款纠纷或留置权被申报。
2. **州法差异巨大**：美国 50 州留置权法各不同，加州、德州、佛州甚至有法定表单格式，用错表单文件无效、不予保护。
3. **阻塞最终拨款**：银行 / 业主拨付最终款项前，GC 必须提交完整 Clean Lien Waiver Package；任何一张文件缺失或金额错误都会推迟收款数周，现金流损失巨大。

---

## AI 怎么做（核心工作流）

1. **项目数据摄入**：对接 Procore / Sage 300 / Excel，导入分包合同清单、SOV（Schedule of Values）、付款证书（Payment Certificate）——确定每个分包商本期应付金额。
2. **自动生成表单**：按项目所在州选择合规模板，预填 GC/Sub 公司名、项目地址、金额、截止日期，生成 PDF，发邮件或短信给各分包商 / 供应商。
3. **智能追踪 + 催签**：实时看板显示哪些已返签、哪些逾期，自动发阶梯提醒（T+3 / T+7 / T+14），对不回复者升级至项目经理预警。
4. **AI 审核 + 打包**：收到扫描件后，OCR 提取关键字段（金额、签名、公证章），与预期值比对，差异自动标红；全部通过后一键导出"Clean Waiver Package" PDF 供业主 / 银行存档。

---

## 护城河

- **50 州法律数据库**：维护各州留置权法定表单 + 规则（金额门槛、公证要求、截止天数），外行难以复制且需持续更新。
- **工作流深度集成**：嵌入 Procore / Sage / Viewpoint 工作流，替换成本极高——PM 团队一旦上手不会换。
- **历史数据飞轮**：处理过的项目越多，AI 对各分包商"惯用错误"（常见金额错填方式）越了解，自动预判并修正。

---

## 市场规模

- 美国持牌总承包商约 **80,000** 家（需管理分包文件）
- 每家平均 5–20 个并行项目，每项目 20+ 个分包关系
- TAM：$80,000 × $3,600/年 ≈ **$288M**（保守）；含二级承包商可达 **$500M+**

---

## 定价

- **$299 / 月**（含 5 个项目 / 50 个分包商追踪节点）
- $499 / 月（无限项目，Procore 深度集成）
- 按项目按需：$79 / 项目（适合小 GC）

---

## 三条件验证

| 条件 | 满足方式 |
|------|----------|
| 输入可数字化 | 合同 / SOV / 付款证书均为结构化数据，可导入或 OCR |
| 过程可描述 | 4 步工作流明确，规则基于州法可编码 |
| 输出可验收 | Clean Waiver Package PDF，字段比对结果明确，零遗漏可量化 |

## 与 GC 主产品的集成路径（2026-02-20 深化）

### 集成场景：从「协调」到「合规」的自动流水线

GC AI 员工团队（主产品）的 Agent 2（分包商协调 Agent）在追踪分包商进度时，
自然会触发付款节点：「XX 分包商已完成本期工作，可以发起付款。」

这个时刻正是 Lien Waiver Coordinator 的最佳触发点：

```
[沟通协调 AI] 分包商完工确认
          ↓ 自动触发
[Lien Waiver AI] 生成本期 Conditional Waiver，发给分包商签字
          ↓ 签字完成
[Pay App 对账 AI] 核对付款申请，确认 Waiver 已收齐
          ↓ 全部通过
[自动] 生成付款批准 + Clean Package 发给业主/银行
```

**这条链路是独特的护城河：** 三个模块分开卖时各值 $200-500/月，但打包卖时形成了一条任何单点竞品都无法复制的工作流。

### 捆绑定价策略

```
GC AI 员工团队（基础沟通协调）：$3,500/月

「GC 财务合规套件」（附加模块）：
  + Pay App 对账审核：$600/月
  + Lien Waiver 追踪协调：$400/月
  ————————————————————
  合计附加：$1,000/月

「全套 GC AI 员工 + 财务合规」：$4,500/月（节省 $500）
```

**销售策略：**
- 阶段一：先卖沟通协调主产品，建立基础信任
- 阶段二（3 个月后）：向现有客户追销财务合规套件
- 逻辑：「我们帮你管好了所有协调电话，现在帮你管 Pay App 和 Lien Waiver？」

## GTM 策略（2026-02-20 更新）

### 切入点 1：发现访谈时增加 Lien Waiver 问题

在发现访谈的痛点确认环节，增加：
> **「你们每个项目有多少个分包商？Pay App 付款前需要收 Lien Waiver 吗？这个流程现在是谁在管？」**

如果 GC 有专职 Project Admin 在管这个，那就是追销机会。

### 切入点 2：独立切入大型 GC（20-50 人规模）

- 大型 GC 可能已经有接线方案（不需要主产品）
- 但 Lien Waiver 管理是他们最头疼的月底工作
- 可以独立以 $299-499/月 切入，积累案例后再推组合方案

### 切入点 3：德克萨斯州法律特殊性（建立差异化）

德克萨斯是少数有「宪法留置权」（Constitutional Lien）保护的州，规则独特：
- 德州要求：分包商必须在 15 天内发出书面通知才能申报留置权
- 德州表单格式：有法定模板，用错表单无效
- **机会：** 专门为德克萨斯 GC 做「德州版 Lien Waiver 合规」，技术壁垒高，竞品几乎没有这个专项
