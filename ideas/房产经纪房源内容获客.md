# 房产经纪 AI 助理——从内容获客到全流程自动化

> **状态更新（2026-02-28）：** 小语（真实潜在客户）昨日与 Kelvin 见面，需求明确，预算 $500-1,000/月。该 idea 从「Exploring」升级为「Active-Pilot」候选。评分从 6.1 → 7.4（有真实客户验证）。

---

## 核心价值主张

独立房产经纪人每天浪费 3-4 小时在重复性工作上（筛房源、协调看房、跟进 lead、填表格），但没有时间也没有预算雇助理。我们用 AI 给他们一个「24/7 在线的数字助理」，让他们专注于真正需要人的工作：建立信任、谈判、Close。

## 产品模块设计

### 模块 1：房源智能推荐（买房侧）
- 接入 MLS 数据 API（Zillow API / RETS / MLSGRID）
- 根据客户偏好（价格区间、学区、户型、通勤时间）自动筛选
- 每日推送「今日新上」「价格调整」给买家客户
- **AI 增值：** 不只是筛选，还生成「为什么推荐这套」的个性化解读

### 模块 2：看房协调机器人（Open House / Showing）
- 自动与 Listing Agent 联系确认看房时间（邮件/短信）
- 与买家客户确认行程，发送提醒
- 看房后自动发送 feedback 问卷，汇总到 CRM
- **时间节省估算：** 每个客户每周节省 1-2 小时

### 模块 3：Lead 自动跟进序列（卖房侧）
- 探房登记 → 自动进入 CRM lead 池
- 7-30 天跟进序列（邮件/短信，个性化变量）
- 未回复 lead 智能重激活
- **转化提升：** 行业平均 lead → 成交 1-3%，自动跟进可提升至 3-5%

### 模块 4：内容获客自动化（小红书/Instagram/WeChat）
- 每周自动生成「本周好房推荐」内容（含图文）
- 市场动态周报（当地均价、成交量趋势）
- 客户成功案例故事（经纪人视角）
- **华人经纪人特供：** 中英双语内容，覆盖华人买家社群
