# 物业/物业管理公司 AI 员工团队（含 HOA 投诉工单）

## 核心价值主张

**"中小型物业管理公司最大的痛点：每天接不完的租客电话。"**

管理 100-500 套房子的物业公司，每天要处理：
- 维修报修（"空调坏了"/"水管漏了"）
- 租金问题（"我这个月付不了/我多付了"）
- 投诉（"邻居太吵/停车位被占"）
- 新租客询问（"还有空房吗？"）
- HOA 违规通知

**一个 PM 经理管 100+ 套房，这些电话把他淹没了。AI 可以接管 80%。**

## 目标客户

- 管理 100-500 套单位的中小型物业公司
- 员工 3-15 人
- 月管理费收入：$30k-$150k
- 痛点：人手严重不足，响应慢导致差评

## AI 员工团队设计

### Agent 1：报修协调 Agent
- 接收维修报修（电话/短信/APP）
- 分类紧急程度（漏水=紧急 / 灯泡=普通）
- 派单给维修工，发送确认
- 跟踪完成，收集租客反馈

### Agent 2：租金问题 Agent
- 到期前 3 天自动提醒（SMS/Email）
- 逾期后自动催款（友好语气）
- 付款异常自动标记，通知 PM 经理

### Agent 3：新租客转化 Agent
- 24/7 接听询房电话
- 回答基本问题（价格/位置/押金）
- 预约看房，发送确认

### Agent 4：HOA 合规 Agent
- 收到违规照片 → 自动生成违规通知
- 跟踪整改期限，发送提醒
- 记录完整合规历史

## 竞品分析

| 竞品 | 做什么 | 差距 |
|------|--------|------|
| AppFolio | 中小 PM 综合管理软件 | 工具，AI 功能弱，不是 Agent |
| EliseAI | AI 租赁代理（大型公寓）| 聚焦 500+ 单元，太贵 |
| BetterBot | AI 聊天机器人 | 只做初始询问，不做工单协调 |

**Renaix 的窗口：中小型 PM 公司（100-500 套），EliseAI 不服务这个市场**

## 商业模式

- Setup：$5,000-$10,000
- 月费：$1,500-$4,000/月（基于管理单元数）
- 定价参考：每套 $3-8/月（100 套 = $300-800/月，看 Agent 配置）
