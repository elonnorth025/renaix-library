# 工程标书合规检查 AI（Bid Compliance）

> **定位：** GC 主产品（沟通协调 AI）的高价值附加模块。
> 不是独立产品切入点，但可以作为扩展功能，提高客单价。

---

## 核心痛点

GC 在提交工程投标（Bid / Proposal）时，必须满足大量合规要求：
- 公立项目（政府合同）：Davis-Bacon Act 工资标准 / 少数族裔企业（MBE）参与要求 / 保险证明 / 许可证状态
- 私人项目：业主要求的特定保险额度 / 分包商资质 / 工期保证

**现状：**
- GC 老板或助理手动检查（需要 2-4 小时/份）
- 容易出错（尤其是政府项目的复杂合规清单）
- 投标失败的第一大原因：**合规文件缺失或格式不对**（而不是价格）

---

## 产品设计

### Agent 功能
1. **文件解析**：上传 RFP（招标文件）→ AI 提取所有合规要求
2. **清单生成**：自动生成「本次投标必须提交的文件清单」
3. **合规检查**：对照已有文件，找出缺失项
4. **模板填充**：常见合规表格（MBE Certification、Insurance Checklist）自动填写
5. **提交前审查**：最终投标包提交前的「合规扫描」

### 工作流
```
上传 RFP → AI 解析要求 → 生成清单 → 上传现有文件 → AI 对比 → 
输出「差距报告」（还需要什么）→ 提醒 GC 准备 → 提交前最终确认
```

---

## 目标客户

**最适合：**
- 同时竞标多个政府项目的 GC（合规要求最复杂）
- 正在扩大规模、第一次接触政府合同的 GC
- 有专职 estimator（预算员）但没有专职合规助理的 GC（50-200 人规模）

**不太适合：**
- 只做私人住宅改建的小 GC（合规要求简单）

---

## 市场规模

- 美国政府建设合同（联邦+州+市）每年约 $500B
- 参与政府投标的 GC：约 10-15 万家
- 每家 GC 平均每年投标 20-50 次
- 合规检查节省时间：2-4 小时/次 × 30 次 = 60-120 小时/年

---

## 商业模式

**作为附加模块（推荐）：**
- 捆绑进「完整版」套餐（月费 +$500-1,000/月）
- 或按使用量收费（$50-100/次投标审查）

**如果作为独立产品：**
- 月订阅：$300-800/月（面向有 estimator 的中型 GC）

## 与 GC 主产品的协同

这个功能和「GC 沟通协调 AI 员工团队」主产品高度协同：
- 同一个 GC 客户，协调 Agent 帮他管现有项目，Bid Compliance Agent 帮他赢得新项目
- 两个产品合并售卖，客单价可以提高到 $4,000-8,000/月
- 共享 GC 行业 Knowledge（建筑术语、合规标准库）

## 风险

| 风险 | 应对 |
|------|------|
| 合规法规频繁更新 | 建立「法规更新监控」Agent，自动更新知识库 |
| GC 不愿上传敏感文件 | 本地部署选项 / 强调数据不离开其系统 |
| 政府合规规定因州而异 | 先聚焦 1-2 个州，再扩展 |
