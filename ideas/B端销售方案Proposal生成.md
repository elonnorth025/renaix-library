# GC 项目 Proposal 生成 AI

> **原始 idea 标题：** B端销售方案/Proposal生成
> **重新定位：** 专注 GC 行业的项目投标提案生成工具，是「沟通协调 AI」的前端漏斗产品

---

## 核心痛点

GC 赢得新项目的第一步是提交 **项目提案（Proposal）** 或 **工程报价单（Bid）**。
这个过程对小型 GC 来说极其痛苦：

- **时间成本：** 一份完整 Proposal（含工程描述、时间表、材料清单、资质证明）需要 4-8 小时
- **质量参差不齐：** 小型 GC 没有专职 estimator，老板自己写 Proposal，往往格式不专业
- **被大 GC 压价：** 大型 GC 有专业 estimator，提案质量更高，小 GC 即使价格更低也容易输
- **转化率低：** 很多小 GC 只写 2-3 个提案/月（因为太费时间），错过大量机会

---

## 产品设计

### 输入（GC 提供）
- 客户需求描述（自然语言：「业主要翻新厨房，约 300 平方英尺，包括橱柜、地板、电气更新」）
- 材料和人工成本估计（可选，有更好；没有也能生成框架）
- 公司资质信息（一次性录入，长期复用）

### 输出（AI 生成）
```
1. 专业格式的 Proposal PDF
   - 项目描述（清晰、专业）
   - 工程范围（详细列出包含/不包含的内容）
   - 时间表（分阶段里程碑）
   - 报价明细（材料 / 人工 / 管理费）
   - 付款计划（首付 / 阶段付款 / 尾款）
   - 公司背景和资质（自动插入）
   - 服务条款（可选）

2. 快速发送选项
   - 直接发 Email 给业主
   - 生成 PDF 供 GC 修改后发送
```

### 特色功能
- **行业标准措辞：** AI 使用 GC 客户熟悉的专业术语（不会显得业余）
- **条款保护：** 自动插入关键保护条款（Change Order 流程、延误不负责条款）
- **本地化定价：** 根据地区调整材料和人工参考价（减少报价差距）

---

## 为什么这是「GC 沟通协调 AI」的好搭档

**场景：** 业主打电话询价 → 沟通协调 AI 接听 → 收集需求 → 自动转给 Proposal 生成 AI → 24 小时内发送 Proposal 给业主

```
业主来电
    ↓
沟通协调 AI 接听（收集需求细节）
    ↓
自动触发 Proposal 生成
    ↓
GC 老板检查/批准（5 分钟）
    ↓
发送给业主（24小时内，比竞争对手快）
```

**这个闭环让 GC 能：**
- 接更多询价（协调 AI 不漏单）
- 更快报价（Proposal AI 自动生成）
- 更高转化率（专业格式 + 快速响应）

---

## 目标客户

**最适合：**
- 每月接到 10-30 个询价请求、但只能回复 3-5 个的 GC（时间限制）
- 没有专职 estimator 的小型 GC（1-10 人）
- 想要提升提案质量、打败大型 GC 的上进型小 GC

**特别适合的子行业：**
- 住宅改造（Kitchen/Bath remodel，提案格式相对标准）
- 商业翻新（Office/Retail renovation，Proposal 更重要）
- 小型新建项目（ADU/Garage，越来越常见）

---

## 市场规模

- 美国住宅改造市场：约 $500B/年
- 参与报价的 GC/承包商：约 50-80 万家
- 平均每月潜在 Proposal：每家 10-30 次
- 时间节省：4-8 小时/份 × 10-30 份/月 = 40-240 小时/月

---

## 商业模式

**按使用量（推荐初期）：**
- $10-20/份 Proposal（低门槛入门）
- 月套餐：$99/月（20 份）/ $199/月（无限）

**捆绑销售（最佳）：**
- 作为「沟通协调 AI 员工团队」的附加模块（+$500-800/月）
- 组合套餐：沟通协调 + Proposal 生成 + Bid 合规检查 = $4,500-6,000/月

## 竞品分析

| 竞品 | 定位 | 差距 |
|------|------|------|
| Jobber | 现场服务软件，有 Proposal 功能 | 不是 AI 生成，需要手动填写 |
| HouseCall Pro | 家庭服务（HVAC/水管），有报价功能 | 不服务 GC，功能浅 |
| Buildertrend | GC 专用，有 Proposal 功能 | 工具，不是 AI；需要大量手动输入 |
| PandaDoc | 通用 Proposal 工具 | 不懂 GC 行业，无行业模板 |
| **Renaix 定位** | GC 专属 AI 生成 Proposal | **深度行业知识 + AI 自动化** |

---

## 风险

| 风险 | 应对 |
|------|------|
| GC 更信任自己写的 Proposal | 设计为「AI 起草，GC 审批」，保留控制感 |
| 报价准确度问题（材料价格变化）| 提醒用户手动确认关键数字；AI 负责格式和措辞 |
| 大型竞品快速跟进 | 深度 GC 行业化是护城河（Jobber 很难快速做 GC 专版）|

## 发现访谈加入问题

把这个 idea 的验证问题加入 GC 发现访谈脚本：
- 「你们每月大概写几份 Proposal？每份需要多少时间？」
- 「你对现在的提案质量满意吗？有没有因为提案格式不好而输掉项目？」
- 「如果可以 30 分钟内生成一份专业的 Proposal，你愿意付多少钱？」
