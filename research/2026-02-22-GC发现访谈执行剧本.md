# GC 发现访谈完整执行剧本
> **创建：** 2026-02-22 | **类型：** GTM 研究 | **优先级：** 🔴 立即执行
>
> **用途：** Kelvin（任川）与 GC 老板第一次接触时的完整操作手册。
> 从冷接触 → 预约 → 访谈 → 信号解读 → 下一步转化，全流程覆盖。

---

## 一、为什么发现访谈是第一优先？

在写一行代码、做一个 Demo 之前，你必须先做 10-20 个发现访谈。

**原因：**
- 你的假设有 80% 概率是错的
- 每一个访谈节省你 2 周开发时间
- GC 老板的话是最好的营销素材（他们的话比你的话更有说服力）
- 第一批客户往往来自访谈对象本身

**成功的发现访谈不是推销，是学习。**
你想知道：他们的世界是什么样的？痛在哪里？现在怎么解决？愿不愿意付钱改变？

---

## 二、目标受访者画像（理想 ICP）

### 首选（最快验证）
- **规模：** 3-20 人的 GC 公司
- **年营收：** $500K - $5M
- **项目类型：** 住宅翻新（Remodeling）/ 新建住宅（Custom Home Build）
- **地域：** DFW（达拉斯-沃斯堡），Kelvin 可面访
- **现状痛点信号：** Google 评论有"slow to respond / no callback"

### 次选（远程验证）
- 任何州的同规模 GC（通过 Facebook 群组接触）
- 西裔 GC 老板（Kelvin 有语言优势）
- 华裔 GC 老板（小圈子，口碑传播快）

### 排除（不访谈）
- 大型 GC（50人以上），他们有专职协调员，不是你的客户
- 纯商业建筑 GC（不同流程，不同决策者）
- 只做装修（Handyman），规模太小，付不起钱

## 三、渠道接触剧本

### 渠道 A：Facebook 群组（量大，最快启动）

**群组列表（DFW 优先）：**
- "Dallas General Contractors & Builders"
- "DFW Contractors Network"
- "Texas Contractors Association"
- "North Texas Building & Remodeling Pros"

**帖子模板（中性，不推销）：**
```
Hey everyone — I'm doing research on how GC companies handle 
the communication side of projects (calls, texts, coordinating subs, 
updating homeowners). 

If you're open to a quick 15-min Zoom call, I'd love to learn 
from your experience. No pitch, no strings attached — just research. 
Happy to share what I learn across the group afterward.

DM me or drop a comment 👇
```

**DM 追进模板（给点赞/评论的人）：**
```
Hey [Name] — thanks for showing interest. 

Quick context: I'm researching how small GC companies manage 
the communication load — calls, sub coordination, homeowner updates. 
Nothing to sell, just genuinely curious to learn from operators 
like you who are in the weeds every day.

15 min on Zoom this week? I'll work around your schedule.
```

### 渠道 B：Google Maps 冷接触（精准，但需要厚脸皮）

**搜索方法：**

  "home remodeling [city]"
  "custom home builder [city]"
```

**过滤标准：**

- 差评关键词：slow response / no callback / hard to reach / communication issues
- 评价数 10-50 条（规模适中，老板还亲自管事）

**SMS 冷短信模板：**
```

```

**回复率预期：** 约 5-10%，每发 50 条得 3-5 个预约

**目标：** Sherwin-Williams Pro 门店 + Home Depot Pro Desk + 独立木材厂

```

**这个渠道的秘密：** Sherwin-Williams Pro Rep 和 GC 老板是多年朋友，他的推荐比任何广告都值钱。

## 四、访谈准备（30 分钟之前）

### 必备信息
- 对方公司名、规模（Google 调研，5 分钟）
- 他们的项目类型（网站/Google Map 看图片）
- 近期评论（关键词：沟通/回复速度/协调）
- 你想测试的 2-3 个假设（今天的访谈要验证什么？）

### 工具准备
- Zoom 链接（提前发给对方）
- Otter.ai / Fireflies.ai 开启录音（征得同意）
- 访谈模板（本文第五部分）打开
- 计时器：访谈结束前 5 分钟提醒自己进入总结

---

## 五、访谈问题完整脚本（45-60 分钟版本）

> **原则：** 多听少说。每个问题之后保持沉默等对方说完。如果对方停了，说"还有吗？"

### 开场（5 分钟）

```
感谢你抽时间。我就是来学习的——你们 GC 的工作我真的不太懂，
所以我所有问题都可能很蠢，请多包容。

我不会在这次聊天里推销任何东西。我就是想弄清楚你们每天的世界
到底是什么样的。

我可以录音吗？只是我自己用来整理笔记，不会对外分享。
```

---

### 模块 1：了解他们的世界（15 分钟）

**Q1. 你现在手上有几个项目同时在跑？**
> *目的：了解规模和复杂度*

**Q2. 一个典型的星期一早上，你第一件事是什么？**
> *目的：了解实际工作流程，而不是他们理想中的流程*

**Q3. 在你的典型工作日里，什么事情占据了你最多时间？**
> *目的：找到最大的时间黑洞*

**Q4. 你通常怎么和分包商保持联系？（电话/短信/WhatsApp/App？）**
> *目的：了解现有工具使用习惯*

**Q5. 你和业主怎么保持更新？他们一般多久问一次进度？**
> *目的：了解业主端的频率和压力*

**Q6. 你有没有专门负责这些沟通的人？还是你亲自处理？**
> *目的：了解是否有协调员，以及他们对这个角色的投入度*

---

### 模块 2：深挖痛点（20 分钟）

**Q7. 在所有这些沟通协调里，什么是让你最头疼的？**
> *目的：开放式挖掘最大痛点*
> *追问：* "能给我讲一个最近发生的具体例子吗？"

**Q8. 有没有过因为沟通/协调问题，导致项目出了问题或损失了钱的情况？**
> *目的：量化痛点，了解财务损失*
> *追问：* "大概损失了多少？""那次事情是怎么发生的？"

**Q9. 下班之后，你还要接工作电话吗？周末呢？**
> *目的：了解生活质量侵占，触发情感层面的痛点*

**Q10. 如果你可以把一件事从你的工作中移走，你会移走什么？**
> *目的：直接定位最大的心理负担*

**Q11. 你现在用什么工具管理项目？（Procore / Buildertrend / 电子表格 / 纸？）**
> *目的：了解技术成熟度和现有工具栈*

**Q12. 这些工具里，哪个你觉得帮助最大？哪个你最不满意？**
> *目的：了解竞品弱点和转换意愿*

---

### 模块 3：测试假设（10 分钟）

> **这里要非常小心，不要变成推销。**
> 描述「已有解决方案」，而不是「你正在做的产品」。

**Q13.（假设测试 - 电话量）**
```
我在和很多 GC 聊，他们说他们每天接 30-50 个电话，
其中大概 30% 是业主问进度、20% 是分包商确认时间。
你们的情况是这样吗？
```
> *目的：验证电话量假设，看他们认不认可这个数字*

**Q14.（假设测试 - 痛点优先级）**
```
在电话/沟通协调这块，有人告诉我他们最烦的是晚上/周末还要接电话，
有人说他们最烦的是分包商不回消息确认时间。
对你来说，哪个更让你头疼？
```
> *目的：帮助你优先哪个痛点*

**Q15.（假设测试 - 解决方案反应）**
```
我了解到有公司开始用 AI 接电话——就是业主打来，AI 先接，
回答常见问题，然后把需要你决策的转给你。
你听说过这种东西吗？第一反应是什么？
```
> *目的：测试对 AI 接线的开放程度和顾虑*

**Q16.（假设测试 - 付费意愿）**
```
假设有一个工具，能帮你处理 70% 的入站电话/短信，让你周末
少被打扰，而且每通电话都有记录——
你觉得这样的东西，如果真的有效，值多少钱一个月？
```
> *目的：不要说一个数字，让他们先说*
> *追问：* "为什么是这个数？""如果是 [你的价格] 呢，你会考虑吗？"

---

### 模块 4：收尾（5 分钟）

**Q17. 你在行业里有没有认识的其他 GC 老板，你觉得他们也会有这方面的困扰？**
> *目的：获取转介绍*

**Q18. 你对这整个话题，有没有什么建议给我？我现在学习阶段，你觉得我哪里理解错了？**
> *目的：获取真实反馈，建立平等关系*

**结束语：**
```
非常感谢，这对我真的很有帮助。

我下一步会整理今天学到的内容，如果你有兴趣，
我可以把总结发给你。

如果我将来真的做出了什么东西，我第一个来找你看。
```

## 六、访谈信号解读

### 🟢 买家信号（继续深入接触）

| 他说什么 | 意味着什么 |
|----------|------------|
| "我每周末都被电话炸" | 高情感痛点，ROI 计算容易 |
| "我现在雇了人专门接电话，但还不够" | 已有预算，替换意愿强 |
| "你能做出来的话告诉我一声" | 潜在试用客户，需要后续跟进 |
| 主动问你定价 | 非常强的信号，进入 Demo 流程 |
| 描述具体的损失金额 | 量化 ROI 完成，关单难度低 |

### 🔴 非买家信号（不要浪费时间）

| 他说什么 | 意味着什么 |
|----------|------------|
| "我们用 Procore，很满意" | 可能规模太大，不是目标客户 |
| "我不相信 AI 能处理这种事情" | 需要大量教育，暂时跳过 |
| "我才 2 个员工" | 规模太小，决策周期长 |
| "我太忙了没时间聊这个" | 但访谈他本身就是验证（他说"太忙"！）|

### 🟡 值得深挖的回答

| 他说什么 | 追问 |
|----------|------|
| "沟通还好，不是我最大的问题" | "你最大的问题是什么？" |
| "我用 Excel/纸质管项目" | "为什么没有用软件？" |
| "我试过 [工具]，不好用" | "哪里不好用？能具体说说吗？" |

## 七、访谈后流程（24 小时内）

### 立即（访谈当天）

1. **写 1 段总结：**  
   > "今天见了 [Name]，他的最大痛点是 ___，意外发现 ___，他对 AI 接线的反应是 ___，买家信号强度：[1-10]。"

2. **更新访谈追踪表：**  
   每次访谈记录到 `pipeline/discovery-tracker.md`（格式见下）

3. **发感谢信息（SMS/Email）：**
   ```
   Hey [Name] — really appreciate the time today. 
   Learned a ton. I'll keep you posted as things develop.
   If you think of anything else, feel free to reach out anytime.
   ```

### 1 周内

4. **如果买家信号强（7+/10）：** 邀请看 Demo
   ```
   Hey [Name] — putting together a quick 20-min demo of what 
   I've been building. Would you be up for a look? 
   Your perspective would be super valuable.
   ```

5. **如果信号中等（4-6/10）：** 加入候补名单
   - 偶尔发现行业内容给他
   - 有案例后重新接触

6. **如果信号弱（1-3/10）：** 记录学习，不浪费时间

## 十、常见心理障碍与破解

**"我还没有产品，凭什么去访谈？"**
> 最好的时机就是没有产品的时候。你没有产品可以推销，
> 受访者完全放松，说的是真心话。

**"GC 老板太忙，不会搭理我。"**
> 他们不是不接触，他们是不接受推销。你是来学习的，不是来卖东西的。
> 区别巨大。

**"我用中文沟通会不会显得不专业？"**
> 用什么语言让对方更舒服就用什么语言。
> 西裔 GC 见到你用西班牙语，感情分立刻+10。

**"万一他问我公司名字或者网站呢？"**
> 准备好："我们是 Renaix，还在早期阶段，现在主要做研究。"
> 不要假装你有完整的产品。真实比包装更值钱。
